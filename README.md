#[freeridesports.com](http://www.freeridesports.com)

1. Install global dependencies
	```
	gem install jekyll && npm install -g grunt-cli && npm install
	```

2. Fire listeners
	```
	grunt dev
	jekyll serve --watch
	```

3. Do work...
	```
	vi <filename>
	git add <filenames>
	git commit
	git push origin master
	```

5. Final build (optional)
	```
	grunt build
	```
